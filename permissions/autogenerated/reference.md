## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-initialize`
- `allow-destroy`
- `allow-save`
- `allow-create-client`
- `allow-load-client`
- `allow-get-store-record`
- `allow-save-store-record`
- `allow-remove-store-record`
- `allow-save-secret`
- `allow-remove-secret`
- `allow-execute-procedure`
